Editor's Note

Malayalam screenwriters have always had to choose between two compromises — write in a tool built for Latin scripts and fight every mixed-script line, or use a generic word processor and lose the formatting that makes a screenplay readable on set.

Scriptty is the third option. The editor speaks screenplay, not "rich text" — every element (Scene Heading, Action, Character, Dialogue, Parenthetical, Transition) has its own behaviour, its own keyboard shortcut, its own typography. Malayalam input is built in (Mozhi · Inscript) so you toggle scripts mid-sentence with Ctrl+Space, never leaving the page.

And when it's time to print, the PDF reads like a proper shooting script — Hollywood single-column or Indian two-column, with title page, scene cards, and a Daily Shoot List your AD can actually use.

INT. THE FEATURE GRID — DAY 02.

Why Scriptty

Built for how screenwriters actually work.

№ 01 — Language

Malayalam & English, mixed freely

Three built-in input schemes — Mozhi (phonetic), Inscript 1, and Inscript 2. No OS keyboard setup needed. Mix Malayalam and English on the same line as casually as you'd switch them in conversation.

№ 02 — Format

Hollywood or Indian two-column

Write in standard Hollywood single-column. Export as Indian two-column for production teams who want dialogue and translation side by side. Page-break-per-scene is one toggle away for both.

№ 03 — Privacy

Fully offline. Yours.

Zero network calls at runtime. Your screenplays live on your machine as .screenplay files. No cloud, no account, no subscription, no telemetry. Works on a flight, on a beach, off-grid.

№ 04 — Three Views

One file, three lenses

Writing view with scene annotations in the margin. Cards view as a typeset shooting-script tear sheet — INT/EXT glyphs, time-of-day tinting, Compact mode, Group-by-location. Story view for Idea, Synopsis, Treatment, and Narrative.

№ 05 — Web Series

Multi-episode, one file

An IDE-style episode explorer with per-episode status (Outline / Draft / Revision / Final), drag-to-reorder, and an Episode Breakout view that drills from a series overview into each episode's scene cards. Export the active episode or the whole series.

№ 06 — Production

Plan the shoot, not just the script

Tag each scene with extras, a location group, and a shoot date. Export a Daily Shoot List PDF — scenes grouped by day & location with industry-standard page-eighths totals. Statistics panel adds Schedule and Episodes views with sortable columns and CSV export.

№ 07 — PDF

An editorial-grade printed page

Title page, prose covers, and scene cards share one masthead vocabulary — small tracked-caps eyebrows, a dominant film title, asterism dividers, Courier-set credits. Per-section page numbering, transition widow control, full-width production cards, Compact card-view export.

№ 08 — Editor

A smart, restrained editor

Auto-uppercase scene headings & characters. Malayalam-aware autocomplete for character names. Smart curly quotes. Adjustable editor font size. Find & replace. Paste a plain-text draft and watch it become a properly formatted screenplay.

№ 09 — Recovery

Autosave that survives a crash

Background autosave to a hidden recovery file as you type. If the app or your machine crashes, the next launch offers to restore the unsaved work — even an unsaved new document survives a power loss. A normal save clears the recovery file.

EXT. THE MANUAL — CONTINUOUS 04.

Reference Card

A pocket guide. For when the page gets crowded.

The Editor

Element navigation

The editor cycles through six element types via Tab and Enter:

  • Scene Heading → Enter → Action
  • Action → Tab → Character
  • Character → Enter → Dialogue
  • Dialogue → Tab → Parenthetical
  • Anywhere → Shift+Enter → new Scene Heading
  • Anywhere → Cmd+Shift+T → Transition

Malayalam input

Ctrl+Space toggles English ↔ Malayalam mid-sentence. Three schemes available — Mozhi, Inscript 2, Inscript 1.

Smart features

Smart curly quotes as you type. Character autocomplete after 2 characters (Malayalam-aware). Auto-uppercase for scene headings and character cues.

Production

Scene cards

Each scene card carries a description, shoot notes, extras list, location group, and shoot date. Cmd+Shift+K opens the Cards view.

Daily Shoot List

From the Export modal, pick Daily Shoot List to emit a per-day production report: scenes grouped by shoot date, sub-grouped by location group, with industry-standard page-eighths totals — ready to hand to the AD.

Statistics

Cmd+Shift+I opens five views — Overview, Characters, Locations, Schedule, Episodes — with sortable columns and CSV export.

Web series

Cmd+Shift+N creates a new series file. Each episode has its own metadata, scene cards, and story sections. Export the active episode or the entire series.

Files & Output

The .screenplay format

Plain JSON. Human-readable. Future-proof. Open it in any text editor if you ever need to.

Export

  • PDF — Hollywood or Indian, with optional Title page, Synopsis, Treatment, Narrative, Scene Cards, Daily Shoot List sections
  • Fountain — plain text screenwriting format, compatible with Highland and Fade In
  • Plain text — formatted .txt with proper indentation

Autosave & recovery

A hidden recovery file is updated as you type. Survives crashes and power loss; cleared on a normal Cmd+S.

What's new.

№ 01

Production planning, end to end

Scene cards carry a location group, shoot date, and extras. Daily Shoot List PDF groups scenes by day & location with industry-standard page-eighths. Statistics panel gains Schedule and Episodes views with sortable columns and CSV export.

№ 02

Editorial-grade PDF redesign

Title page, prose covers, and scene-card / shoot-list openers share one masthead vocabulary — eyebrow, dominant title, asterism, Courier-set credits. Per-section page numbering. Transition widow control. Courier Prime now bundled into every PDF.

№ 03

Episode Breakout view

For series projects: a top-level card per episode with a scene preview list. Click an episode to drill into its scenes. IDE-style explorer with per-episode status, drag-to-reorder, hover-rename.

№ 04

Paste to Screenplay

Drop plain text into the editor and Scriptty converts it into a properly formatted screenplay — Hollywood-style detection plus a Malayalam-aware character-cue path.

№ 05

Autosave & crash recovery

Background autosave to a hidden recovery file. Survives crashes and power loss. A normal save clears it. Even an unsaved new document doesn't disappear if the lights go out.

№ 06

Smart curly quotes & adjustable editor type

Straight quotes turn curly as you type, in both Latin and Malayalam contexts. Editor font size is now adjustable from Settings — independent of the printed PDF body size.
